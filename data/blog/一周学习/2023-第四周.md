### [10 powerful ways to use CSS variables](https://nerdy.dev/custom-prop-categories)

```css
:root {
  --adaptive-padding: 0.5rem;

  @media (width >= 320px) {
    --adaptive-padding: 1rem;
  }
  @media (width >= 720px) {
    --adaptive-padding: 1.25rem;
  }
  @media (width >= 1024px) {
    --adaptive-padding: 1.5rem;
  }
}
```
